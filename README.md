# 🎮 PROA - Eu Escolho Você

Este projeto é um protótipo simples de um aplicativo que sugere jogos de forma aleatória, desenvolvido durante o curso PROA.

A ideia é ajudar o usuário a decidir rapidamente qual jogo jogar.

## 💡 Como funciona

O aplicativo possui uma interface simples:

- Um botão com a opção **"Me dê um jogo!"**
- Ao clicar, o sistema exibe:
  - A imagem do jogo
  - O nome do jogo escolhido aleatoriamente
