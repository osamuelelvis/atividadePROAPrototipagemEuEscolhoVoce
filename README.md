# 🎨 PROA - Prototipagem: "Eu Escolho Você"

Este projeto consiste na prototipagem de uma aplicação desenvolvida durante o curso PROA, com foco em experiência do usuário (UX) e design de interfaces (UI).

## 📚 Sobre o projeto

A proposta foi criar um protótipo funcional de uma aplicação, aplicando conceitos de:

- Design de interface (UI)
- Experiência do usuário (UX)
- Organização de fluxo de navegação
- Estruturação visual de telas
